## Sistema de Matrícula Digital

Recursos disponíveis no sistema:

- Cadastro e listagem de Cursos
- Cadastro e listagem de Alunos
- Matrículas de Alunos em Cursos


## Instalação

- Instalar o laravel via composer
- Fazer o clone do repositório
- Ajustar o arquivo .env com dados de acesso ao DB (MySql)
- Criar DB com nome 'db_ensino' 
- Executar as migrations
